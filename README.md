## 개발 환경
  * 기본 환경
    * OS : Mac OS X
    * IDE : Android Studio 4.0.1
    * JVM target : Java-1.8
    * Kotlin 1.3.72
  
  * 코어 라이브러리
    * Androidx 1.3.1
    * Coroutine 1.3.8
    * MultiDex 2.0.1
    * Lifecycle 1.3.8
    * Room 2.3.0-alpha01
    * Paging 3.0.0-alpha05

  * 기타 라이브러리
    * Dagger 2.27
    * Retrofit 2.8.1
    * OkHttp 4.7.0
    * Glide 4.11.0
    * Timber 4.7.1
    * Logger 2.2.0
    
  * 이미지 출처
    * 앱 아이콘 : 자체 제작
    * 기본 아이콘 : MATERIAL DESIGN
    * 카카오 캐릭터 : 카카오 프렌즈 월페이퍼
